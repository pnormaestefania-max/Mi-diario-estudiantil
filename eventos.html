Eventos en JavaScript

Evento Descripción Ejemplo de Uso
onclick Se activa cuando el usuario hace clic sobre un elemento. <button onclick="alert('Hola!')">Haz clic</button>
onmouseover Se activa cuando el puntero pasa por encima de un elemento. <div onmouseover="this.style.background='yellow'">Pasa el mouse aquí</div>
onkeydown Se activa cuando el usuario presiona una tecla. <input onkeydown="console.log('Tecla presionada')">
onload Se ejecuta cuando la página o un recurso ha terminado de cargarse. <body onload="alert('Página cargada')"></body>
onchange Se ejecuta cuando cambia el valor de un campo de formulario. <input type="text" onchange="alert('Texto cambiado')">
